# 🌐 GlobalTrade Hub Server

🔗 Client Live Site: https://global-trade-hab.web.app

🔗 Server Live Site: https://global-hub-server.vercel.app/

## 📖 Description

This is the backend server for GlobalTrade Hub. It handles API requests, authentication, and database operations for managing products, imports, and exports.

---

## 🚀 Features

* 🔐 JWT Authentication (if used)
* 📦 Product CRUD Operations
* 📥 Import Product Logic
* 📤 Export Product Management
* 🔄 Real-time quantity update using MongoDB operators

---

## 🛠️ Technologies Used

* Node.js
* Express.js
* MongoDB
* Firebase Admin (optional)

---


## 🔐 Environment Variables

```env
PORT=5000
DB_USER=your_db_user
DB_PASS=your_db_pass
```

---

## 📡 API Endpoints (Example)

* GET /products
* POST /products
* PUT /products/:id
* DELETE /products/:id
* POST /imports
